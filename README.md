# Mealix - Diet Planner AAA

¡Bienvenido a Mealix! Esta es una aplicación de planificación de dietas con una interfaz premium, diseñada para ofrecer la mejor experiencia de usuario.

## 🚀 Despliegue Rápido

Este proyecto está configurado para desplegarse fácilmente en **Netlify**.

### Pasos para Netlify:
1. Conecta tu cuenta de GitHub a Netlify.
2. Selecciona este repositorio (`Mealix`).
3. Netlify detectará automáticamente el archivo `netlify.toml` y configurará el build:
   - **Base directory**: `diet-planner`
   - **Build command**: `npm install && npm run build`
   - **Publish directory**: `dist`

## 🛠️ Tecnologías
- **Vite + React**
- **Framer Motion** para animaciones fluidas.
- **Tinder Cards** para el intercambio de comidas.
- **Diseño AAA** con estética premium.

## 📂 Estructura
- `/diet-planner`: El código fuente de la aplicación.
- `/project-specs`: Documentación del proyecto.
- `netlify.toml`: Configuración de despliegue.
