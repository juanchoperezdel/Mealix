# Diet Planner Web App - Task List

### [ ] Task 1: Initialize Vite/React Project
- Create a new Vite React application in `src/`.
- Ensure Vanilla CSS architecture is chosen with a clear `index.css`.
- Remove default boilerplate.

### [ ] Task 2: Create Design System (ArchitectUX)
- Define CSS custom properties (variables) for a premium color palette (e.g. vibrant gradients, sleek dark mode variables).
- Define typography setup (Inter, Roboto, or Outfit).
- Add micro-animations utility classes.

### [ ] Task 3: Build User Input Interface
- Implement an aesthetic, multi-step or comprehensive form.
- Form fields: Primary Goal, Monthly Budget, Allergies, Dietary Preferences, Meals per day.
- Add form validation and loading state UI.

### [ ] Task 4: Implement AI Integration Service
- Create an API service file `aiDietService.js`.
- Construct a robust prompt structure that takes user inputs and outputs a strictly formatted JSON (30 days, exact meals, recipes/preparation).
- Mock or configure actual API call based on provided keys.

### [ ] Task 5: Build Diet Plan Display
- Create components to render the 30-day diet.
- Add weekly navigation and daily recipe cards.
- Ensure the presentation is stunning and functional.

### [ ] Task 6: Final QA & Polish
- Check edge cases.
- Validate that design looks "premium" and micro-interactions work perfectly.
